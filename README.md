# KAC-PPI Dashboard

> **2026 국토교통 데이터활용 경진대회 — 제품·서비스 개발 부문**
> 공항 VOC 빅데이터 기반 공항별 페인포인트 진단지표(KAC-PPI)와 데이터 기반 서비스 개선 처방 시스템

## 핵심 산출물

| 산출물 | 설명 |
|---|---|
| `app.py` | Streamlit 4페이지 대시보드 (시제품 본체) |
| `analysis.py` | PPI 산출 파이프라인 (CLI도 가능) |
| `recategorize_voc.py` | VOC '기타' 23.8% LLM 재분류 (AI학습도구 가점) |
| `fetch_external.py` | 외부 공공데이터 수집 (데이터융합 가점) |
| `data/ppi_table.csv` | 공항 × 카테고리 PPI 점수표 |

## 실행 방법

```bash
# 1) 의존성 설치
pip install -r requirements.txt

# 2) PPI 사전 계산 (선택 — app.py가 자동 계산함)
python analysis.py

# 3) Streamlit 대시보드 실행
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속.

## KAC-PPI 산출식

```
PPI(공항 a, 카테고리 c) =
    0.4 × 불편불만 상대 점유율 (vs 전체 평균)
  + 0.3 × 1만명당 정규화 건수 (공항 규모 보정)
  + 0.2 × 시계열 증가율 (최근 12개월 vs 이전 12개월)
  + 0.1 × 감정 가중치 (불편 / (불편+칭찬+제안))
→ 카테고리 내 0~100 스케일로 정규화
```
